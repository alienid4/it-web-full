📖 圖例與名詞解釋
節點: A 級 (金) B 級 (銀) C 級 (銅) 未確認 故障/受影響 線型: ━ 手動連線 (信任) ┄ 自動偵測 (待確認)
A/B/C 級別 (tier):業務重要性分類。 A 級 (金) = 核心關鍵業務,故障立即搶救; B 級 (銀) = 重要業務,幾小時內處理; C 級 (銅) = 一般系統,當天處理就好。
深度:點選某節點後重整圖,以該節點為中心向外擴展幾層鄰居 (BFS)。深度 1 = 只看直接相連,深度 2 = 再往外一層,以此類推。沒選中節點時不生效 (顯示全圖)。
顯示模式: 📦 系統別 (業務系統節點) / 🖥️ 主機 (hostname 節點) / 🌐 IP (192.168.x 節點)。同一份採集資料,三種視角切換看。
