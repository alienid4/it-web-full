## 步驟 4：MongoDB 驗證

```bash
mongosh inspection --eval '
db.vmware_snapshots.find({}, {
  timestamp: 1,
  "vcenter.label": 1,
  "vcenter.ip": 1,
  status: 1,
  "about.version": 1,
  cluster_count: { $size: "$clusters" },
  host_count:    { $size: "$hosts" }
}).sort({timestamp: -1}).limit(10).pretty()
'
```

預期看到 5 筆 success snapshot，每個 cluster/host count 對得上 VMware 月報。

驗 index 也建好：

```bash
mongosh inspection --eval 'db.vmware_snapshots.getIndexes()'
# 預期看到 timestamp_-1_vcenter.ip_1 + ttl_180d
```

## 異常處理

### 5 VC 部分失敗

例：板橋 OK，VCF fail。snapshot 還是會寫進 MongoDB（status=fail），UI 那個 chip 會變橘色「pending」。然後：

```bash
# 看單台 log
sudo -u sysinfra python3 /opt/inspection/collector/vcenter_collector.py --only VCF -v
```

### MongoDB 沒寫進去

確認 mongod 跑著：`systemctl status mongod`  
確認 collector 用的是 localhost:27017：collector 預設 hardcode `MongoClient("localhost", 27017)`

### UI 還顯示 mock

- 確認 `mongosh inspection --eval 'db.vmware_snapshots.countDocuments({})'` > 0
- 重啟 itagent-web：`sudo systemctl restart itagent-web`
- 強制瀏覽器 reload (Ctrl+Shift+R)
- 看 `/api/vmware/overview` JSON response 有沒有 `_source: "inline_mock_fallback"`
