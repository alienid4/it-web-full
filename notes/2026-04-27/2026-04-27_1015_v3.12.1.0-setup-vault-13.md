# 2026-04-27 10:15 — 13 上設定 vcenters.yaml + ansible-vault

目的：填 5 個 vCenter 真 IP + 加密 administrator 帳密，給 collector 用。

**前置**：v3.12.1.0-vmware-collector 已套（即 `/opt/inspection/data/vmware/` 已建）。

**預期耗時**：5 分鐘

---

## 步驟 1：建 vcenters.yaml（填 5 VC 真 IP）

```bash
cd /opt/inspection/data/vmware
sudo cp vcenters.yaml.sample vcenters.yaml
sudo vi vcenters.yaml
```

填入（IP **以你實際環境為準**，不寫進 notes）：

```yaml
vcenters:
  - label: "板橋"
    ip:    "<板橋_VC_IP>"
  - label: "內湖-1"
    ip:    "<內湖-1_VC_IP>"
  - label: "內湖-2"
    ip:    "<內湖-2_VC_IP>"
  - label: "VCF"
    ip:    "<VCF_VC_IP>"
  - label: "敦南"
    ip:    "<敦南_VC_IP>"
```

存檔後改權限：

```bash
sudo chmod 600 /opt/inspection/data/vmware/vcenters.yaml
sudo chown sysinfra:itagent /opt/inspection/data/vmware/vcenters.yaml
```

驗證：

```bash
sudo -u sysinfra cat /opt/inspection/data/vmware/vcenters.yaml
# 應該讀得到; ls -la 應該看到 -rw-------
```

---

## 步驟 2：建 vault 加密 vc_credentials

確認 `.vault_pass` 存在（既有；它原本就被 inspection 系統其他 role 用）：

```bash
sudo ls -la /opt/inspection/.vault_pass
# 預期: -r-------- 1 sysinfra ...
```

建加密檔：

```bash
cd /opt/inspection/data/vmware
sudo ansible-vault create vc_credentials.vault \
  --vault-password-file /opt/inspection/.vault_pass
```

編輯器打開後填：

```yaml
user: "administrator@vsphere.local"
password: "<真實密碼>"
```

存檔後加權限：

```bash
sudo chmod 600 /opt/inspection/data/vmware/vc_credentials.vault
sudo chown sysinfra:itagent /opt/inspection/data/vmware/vc_credentials.vault
```

驗證可解密：

```bash
sudo -u sysinfra ansible-vault view /opt/inspection/data/vmware/vc_credentials.vault \
  --vault-password-file /opt/inspection/.vault_pass
# 應該看到剛才填的 user/password
```

## 步驟 3：檔案盤點（最終狀態）

```bash
sudo ls -la /opt/inspection/data/vmware/
```

預期：

```
-rw-r--r--  README.md
-rw-------  vc_credentials.vault   (sysinfra:itagent, $ANSIBLE_VAULT;1.1;AES256 開頭)
-rw-------  vcenters.yaml          (sysinfra:itagent)
-rw-r--r--  vcenters.yaml.sample
```

---

## 資安要點

- `vcenters.yaml` 含實 IP → 600 不外洩
- `vc_credentials.vault` ansible-vault AES256 → 600 + 走 `.vault_pass`
- `.vault_pass` 從來不 commit（跟既有 vault 共用密碼）
- collector 程式碼**嚴格 read-only**（見 `vcenter_collector.py` 開頭註解 + `feedback_vmware_admin_readonly.md`）
- 即使用 administrator 帳號也只 RetrieveContent + 讀屬性，絕不 Create/Modify/PowerOn/Off
