# 2026-04-24 12:30 — v3.11.26.0 深度檢查版本對照

**做了什麼**：查看任一深度檢查報告後，可挑另一個版本**並列對照**，一眼看出哪個面向有變化。

## 在 13 上套

```bash
cd /tmp
rm -rf v3.11.26.0 patch_v3.11.26.0.tar.gz 2>/dev/null
curl -fLO https://github.com/alienid4/it-web-full/releases/download/v3.11.26.0/patch_v3.11.26.0.tar.gz
tar xzf patch_v3.11.26.0.tar.gz
cd v3.11.26.0
sudo bash install.sh
```

## 使用

1. 點「🔍 深度檢查」→ 從歷史點一個版本查看
2. 完成畫面底部下載按鈕旁新增 **「📊 對照其他」**（藍色）
3. 點它 → 藍框選擇器列出該主機其他歷史版本
4. 選一個 → **並列對照表**：

```
┌──────────────────────────────────────────────┐
│ 📊 版本對照            ▲ 整體惡化             │
├──────────────────────────────────────────────┤
│ 左 (舊)              │ 右 (新 / 當前)          │
│ 2026-04-24 10:29     │ 2026-04-24 11:23       │
│ 警告 · P=6 W=2 F=0  │ 警告 · P=5 W=3 F=0     │
├──────────────────────────────────────────────┤
│ 面向     舊         新         變化           │
├──────────────────────────────────────────────┤
│ [1/9] 效能   ✅PASS   ✅PASS     —            │
│ [2/9] 頻寬   🟡WARN   🟡WARN     —            │
│ [4/9] Session ✅PASS  🟡WARN   ▲ 惡化 🟡       │
│ [6/9] 時間   🟡WARN   🟡WARN     —            │
│ ...                                          │
├──────────────────────────────────────────────┤
│ [✕ 關閉對照]                                  │
└──────────────────────────────────────────────┘
```

**變化的列會自動黃底高亮**，不用一行一行讀。

## 情境

- **客訴調查**：客訴當下跑一次，跟昨天同時段比 → 看哪個面向變了
- **驗證修復**：調完 kernel 參數後 → 對照前後差異 → 確認 conntrack 真的降下來
- **事後復盤**：重啟服務後 → 對照前後 → 看影響面
