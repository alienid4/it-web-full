## 一次貼這一整塊（5 顆串起來）

每顆自動備份 → 複製 → 重啟 → 驗證。若任一顆 FAIL 會停。

```bash
cd /tmp
set -e

for V in v3.11.23.0 v3.11.23.1 v3.11.24.0 v3.11.25.0 v3.11.26.0; do
    echo ""
    echo "═══════════════════════════════════════════════"
    echo "   套用 $V"
    echo "═══════════════════════════════════════════════"
    rm -rf "$V" "patch_${V}.tar.gz" 2>/dev/null
    curl -fLO "https://github.com/alienid4/it-web-full/releases/download/${V}/patch_${V}.tar.gz"
    tar xzf "patch_${V}.tar.gz"
    cd "$V"
    sudo bash install.sh || { echo "❌ $V 套用失敗, 停下"; exit 1; }
    cd ..
    sleep 2
done

echo ""
echo "═══════════════════════════════════════════════"
echo "   全部完成, 驗證版本"
echo "═══════════════════════════════════════════════"
cat /opt/inspection/data/version.json | python3 -c 'import sys,json; d=json.load(sys.stdin); print("版本:", d["version"]); print("更新時間:", d["updated_at"])'
```

## Rollback 全退（不太可能需要）

每顆 patch 都在 `/var/backups/inspection/pre_vX.Y.Z.W_<TS>/` 存了備份。若要全退：

```bash
# 從最新的 26.0 往回退
for V in v3.11.26.0 v3.11.25.0 v3.11.24.0 v3.11.23.1 v3.11.23.0; do
    BACKUP=$(ls -td /var/backups/inspection/pre_${V}_* 2>/dev/null | head -1)
    [ -z "$BACKUP" ] && { echo "無 $V 備份, 跳過"; continue; }
    echo "退回 $V → 復原自 $BACKUP"
    # 看備份內有哪些檔, 手動 cp -p 回去
    ls "$BACKUP"
done
sudo systemctl restart itagent-web
```

（建議不要全退，各顆 install.sh 末段印了各自 rollback 指令，需要再個別處理。）
